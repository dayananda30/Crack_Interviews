A text file is a computer file that only contains text and has no special formatting such as bold text,
italic text, images, etc. With Microsoft Windows computers text files are identified with the .txt file
 extension, as shown in the example picture to the right.
An example of a text file and ASCII Art can be seen in Kirk text file. You can click this link to open the
.txt file in your browser or right-click the file to save the text file to your computer.
